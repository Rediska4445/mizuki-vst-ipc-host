=== VST/VST3 IPC Host ===

java -cp  rf.ebanina.vst.Host [options]

Launch options:
  -port, -p       TCP server port (default: 888)
  -host, -h       IP to bind to (default: localhost)
  -max-channels  Max audio channels (default: 8)
  -max-samples  Max samples per channel (default: 4096)
  -mode, -m       Processing mode (default: plugin)
  -quiet, -q            Quiet mode (minimal logs)
  -verbose, -v          Verbose logs (default)
  -help, --help         Show help

Processing modes (-mode):
  • echo     - Pass-through without changes
  • silence - Silence (zero samples)
  • gain    - Gain x2 (clipping possible)
  • invert  - Phase inversion (180°)
  • delay   - Echo/delay (~50ms)

  • distort - Soft distortion

  • plugin  - VST/VST3 plugin (default)

Examples:

  # Basic launch (localhost:888, plugin mode)

  java -cp Mizuka.jar rf.ebanina.vst.Host

# Network access (all interfaces)

  java -cp Mizuka.jar rf.ebanina.vst.Host -host 0.0.0.0 -port 999

# Professional audio (16 channels, 8192 samples)

  java -cp Mizuka.jar rf.ebanina.vst.Host -max-channels 16 -max-samples 8192

# Test mode without plugin (echo)

  java -cp Mizuka.jar rf.ebanina.vst.Host -mode echo

IPC Protocol (TCP):

  • Commands: INIT_PLUGIN, PROCESS_AUDIO, SET_PARAMETER, GET_PARAMETER, etc.

  • Audio: float32, sample-major order (sample→channel)

  • Format: [length:int32][command+parameters][audio data]

Supported commands:

  INIT_PLUGIN(type;path;sr;bs;rt;double)     Load VST/VST3

  PROCESS_AUDIO(channels;samples;framesRead) Process audio

  SET_PARAMETER(idx;value)                   Set parameter

  GET_PARAMETER(idx)                         Read parameter

  GET_PARAM_COUNT                            Parameter count

  OPEN_EDITOR                                Open GUI

  SAVE_STATE                                 Save preset

  LOAD_STATE(base64)                         Load preset

  TERMINATE_PLUGIN                           Terminate plugin

  HELP                                       List of commands

Logs:

  • ipc-out.log - information

  • ipc-err.log - errors

Notes:

  • VST/VST3 autodetection by file extension

  • VST plugin editor GUI support

  • State saving/loading (Base64)

  • Graceful shutdown on Ctrl+C
